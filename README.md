# README

### 開発言語
- Ruby3.0.1
- Rails6.1.7

### 就業Termの技術
- devise
- AWS EC2

### カリキュラム外の技術
- Vue.jsを用いたSPA

### 実行手順
```
$ git clone git@github.com:tomoki4321/okan_task.git
$ cd okan_task
$ bundle install
$ yarn install
$ rails db:create && rails db:migrate
$ rails s
```
